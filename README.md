## Getting Started

```bash
# Clone this project
$ git clone https://github.com/trpiano/links

# Access
$ cd links

# Install dependencies
$ yarn
# Or
$ pnpm i
# Or
$ npm i

# Run the project
$ yarn expo start
# Or
$ pnpx expo start
# Or
$ npx expo start

# The server will initialize in the <http://localhost:8081>
```
